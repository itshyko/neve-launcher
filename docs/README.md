# Neve Launcher

Le launcher minecraft du projet Neve Aventure par HyKo.dev

## Installation

Télécharger le launcher dans les "releases" en sélectionnant votre plateforme.

1. Ouvrez le fichier.
2. Connectez-vous à votre compte Minecraft.
3. Cliquez sur Jouer!
